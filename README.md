# YelpCamp
•	In this Website the user can give its free opinion about the camping grounds he/she has/had visited.
•	The user can add, edit, delete his/her campground and rate or leave a review for other campgrounds by making an account on website.
•	It uses MAPBOX API to show the location of campgrounds.
•	Styling done with popular CSS framework Bootstrap 5
•	 Authentication performed with extremely flexible authentication middleware for Node.js -Passport
•	Deployed on cloud platform supporting Node.js - Heroku
•	I was responsible for adding the maps to show the location of campgrounds, design the schema and model of campgrounds, users and reviews using mongoose, addition of middleware and creating web pages of basic CRUD operations on Visual Studio Code
